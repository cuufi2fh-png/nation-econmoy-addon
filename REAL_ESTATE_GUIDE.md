# 부동산 거래 시스템 가이드

## 개요
부동산 사장 NPC를 통해 부동산을 중개하는 시스템입니다. 주택, 주택 임대, 상업용 건물을 거래할 수 있습니다.

## 부동산 등록 방법

### 1. 부동산 지팡이(Property Wand) 사용
- **아이템**: 부동산 지팡이
- **방법**: 
  1. 부동산 지팡이를 손에 들고
  2. 첫 번째 블록을 클릭 (일반 클릭)
  3. 두 번째 블록을 클릭 (웅크리고 클릭)
  4. 지팡이를 들고 다시 클릭하면 UI가 나타남

### 2. 부동산 등록 UI
- **용도별 거래 방식 선택**:
  - 🏠 주택 매매 등록
  - 🏠 주택 월세 임대 등록
  - 🏢 상업용 건물 통째로 매매
  - 🏢 상업용 건물주 직접 등록 (방 임대업)

- **필수 정보 입력**:
  - 매물/건물 이름
  - 가격 / 월세 (₩)

### 3. 부동산 표지판(Sign) 생성
등록 후 자동으로 부동산 영역의 중앙에 표지판이 생성됩니다.

## 부동산 거래 프로세스

### 표지판 클릭 → NPC 소환
1. 부동산 표지판을 우클릭
2. 부동산 사장 NPC가 나타남
3. NPC를 클릭하여 거래 시작

### 거래 UI
**주택 매매/건물 매매의 경우:**
- 물건명, 가격, 수수료(5.14%), 총액, 판매자 정보 표시
- "구매" 또는 "인수" 버튼으로 거래 완료

**주택 임대의 경우:**
- 물건명, 월세, 임대 기간(48분마다), 집주인 정보 표시
- "임대 계약" 버튼으로 계약 완료

## 수수료 및 세금 체계

### 거래 수수료
- **비율**: 거래가격의 5.14%
- **부담**: 구매자가 추가로 지불
- **수령**: 서버 중앙계좌

### 국가 세금
- **비율**: 국가마다 다름 (기본 50%)
- **대상**: 판매자
- **계산**: 거래가격 × 국가세율
- **수령**: 국가 중앙은행

### 거래금 흐름 예시
**판매가: 100,000₩인 경우**
- 구매자 결제: 105,140₩ (100,000 + 5,140)
- 판매자 수령: 50,000₩ (100,000 - 50% 세금)
- 서버 계좌: 5,140₩ (수수료)
- 국고: 50,000₩ (국가세)

## 부동산 증명서(Building Cert)

### 증명서 발급
- 부동산 매매 거래 시 자동 발급
- 당신의 소유권을 증명하는 아이템

### 증명서 사용
1. 증명서를 손에 들고 우클릭
2. 당신의 부동산 목록 표시
3. 부동산 선택 시 상세 정보 확인:
   - 물건명
   - 현재 소유자
   - 기본 가격
   - 거래 기록 (최근 3건)

## 주요 기능

### 주택 매매
- 단독 소유권 판매
- 수수료 및 세금 자동 계산
- 소유권 자동 이전

### 주택 임대
- 월세 임대 (48분마다 자동 출금)
- 세입자 자동 추가/해제
- 수입금 자동 입금

### 상업용 건물 관리
- 건물 통째로 매매
- 방 임대업 시작
- POS 단말기 자동 제공 (점포 임대 시)

## 주의사항
- 거래 후 수수료와 세금은 되돌릴 수 없습니다
- 임대료는 48분(실시간 기준)마다 자동 출금됩니다
- 잔액 부족 시 거래가 불가능합니다
- 국가 세율은 국가 지도자가 변경할 수 있습니다
